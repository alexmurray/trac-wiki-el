* About trac-wiki-el

Emacs mode for editing Trac wiki pages, tickets, blog posts, and more,
with colour syntax highlighting for Trac wiki formatting markup.

Originally developed by Shun-ichi GOTO <shunichi.goto@gmail.com>.
